# Contributing

## Setup

1. Install dependencies
   - `{{INSTALL_COMMAND}}`

2. Verify tests pass
   - `{{TEST_COMMAND}}`

## Development Workflow

- Make small, reviewable changes.
- Add/update tests for behavior changes.
- Keep formatting/linting clean.

## Commands

- Test: `{{TEST_COMMAND}}`
- Lint: `{{LINT_COMMAND}}`
- Format: `{{FORMAT_COMMAND}}`

## CI

CI should run at least:
- tests
- lint (or format check)

Keep local commands aligned with CI.
